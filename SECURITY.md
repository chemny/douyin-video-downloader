# Security

- Authentication state is local-only. Do not commit browser profiles, cookies, storage-state files, SQLite databases, or downloaded media.
- `collect-user` uses a local browser profile and only asks the user to scan a QR code when the current Douyin login state is unavailable.
- This project does not include login bypass, anti-bot bypass, or credential collection.
- Downloaded media should be handled according to Douyin platform rules and applicable copyright laws.
- If you find a security issue, do not publish secrets or cookies in a public issue.
